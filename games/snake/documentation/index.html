How to use without options bar and/or on another Web page

The game can easily be used without the bar with the settings.
 
In order to embed the game on another Web page, you need to connect the script executable files to HTML file for your Web page. To do this, insert the <script src="%path%/libcanvas.js"></script> <script src="%path%/application.js"></script> between the tags <head></head>.  %path% is the path of the web page to files.
Make sure you have choosen a right path for the files.
 
Then write the code below where you want to have a game board:
 <script> initGame(null, document.getElementById('movecount'), document.getElementById('stopwatch')); </script>
This can be in any unit like <div></div>
 
Also you need a place to use counter of moves and stopwatch if you want:
<span id="movecount">0</span>
<span id="stopwatch">00:00:00</span>
<span id="boardsize">10 x 10</span> — show game board size
<div id="uncommonfoodcontainer"><li><b id="button" class="button"><span id="uncommonfood"></span></b></li></div> — uncommon food timer

The list of functions for the buttons

To add function to button you can add onclick = "%Function_name%();" to button description.

For example: <a href="#" class="button left" onclick = "undo();"></a>

Here all functions of game:

undo(); — undo last move

redo(); — redo last move that was undo

pauseGame(); — pause or rresume game

changePieceSize(true); — increase the size of every square

changePieceSize(false); — decrease the size of every square

changeLevel(true); — to level up

changeLevel(false); — to level down

To use game messages you should write code below:

onclick ="message(true,'Your_text')"; — be accurate about special symbols
